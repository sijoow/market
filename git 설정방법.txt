git init repository 생성해줘라
기본 브랜치 main으로 변경
git branch -M main
git add .
git commit -m 
원경 저장소 추가하기 
git push -u https://github.com/sijoow/yogico.git main
코드를 추가했을땐
변수 문법 사용하기
git remote add origin https://github.com/sijoow/yogico.git 
git push -u  origin main 
git push 로만 입력해주면 이제 업로드 됨

수정하는 방법
git add .
git commit 수정했다
git push 해주면 수정됨
